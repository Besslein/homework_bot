<h1>Описание</h1>
Телеграм-бот для проверки статуса домашнего задания.
<hr>
<h1>Функционал</h1>
Проверяет статус домашнего задания
<hr>

<h1>Установка и запуск</h1>
<h2>Клонировать репозиторий:</h2>
<hr>
<code>git clone <repository_name></code>
<hr>
<h2>Перейти в папку проекта:</h2>
<code>cd .../homework_bot</code>
<hr>
<h2>Создать виртуальное окружение:</h2>
<code>python -m venv venv</code>
<hr>
<h2>Активировать виртуальное окружение:</h2>
<code>source venv/Scripts/activate</code>
<hr>
<h2>Установить библиотеки:</h2>
<code>pip install -r requirements.txt</code>
<hr>
<h2>Запустить бота:</h2>
<code>python homework.py</code>
